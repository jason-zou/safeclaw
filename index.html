安全不是 prompt 的事
是运行时的事

传统 AI 工具仅靠 prompt 约束安全行为——这既脆弱又不可靠。SafeClaw 通过运行时 + prompt 双重保障，在代码层面强制执行安全策略。

每条消息都经过入站/出站双向安全管线检测：Prompt 注入防护、PII 脱敏、凭证泄露扫描、危险操作确认——全部在 Rust 运行时中完成，不依赖模型"自觉"。
